Player

class Player (View source)

The Player class represents a player in the game. This includes co-op players.

It does not include players who joined the lobby but didn't launch into the actual game.

Properties

string $name The player's name.
int $index The player's index.
int $number The player number. This is different from the index. The player number is always a sequential unique ID. The player index is the same for multiple players in the case of a co-op game.
bool $human Defines if the player is a human.
bool $spectator Defines if the player is a spectator.
int $teamIndex Defines player's team index (0 = no team).
bool $owner Defines if player is an owner of the game.
int $civId ID of the player's civilization.
int $colorId Player color ID.
bool $isCooping Indicates if the player is cooping in the game.
int $feudalTime Player's feudal time (in ms, 0 if hasn't been reached).
int $castleTime Player's castle time (in ms).
int $imperialTime Player's imperial time (in ms).
int $resignTime Player's resign time (in ms) or 0 if player hasn't resigned.
InitialState $initialState Contains the player's initial state, such as starting resources and population.
